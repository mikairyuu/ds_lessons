ФУНКЦИЯ ДобавитьСтолбецЕдиниц(X):
    m = количество строк в X
    столбец_единиц = вектор из m единиц
    X_new = [столбец_единиц | X]
    ВЕРНУТЬ X_new
КОНЕЦ ФУНКЦИИ

ФУНКЦИЯ ЛинейнаяРегрессия(X, y, learning_rate, num_iterations):
    X = ДобавитьСтолбецЕдиниц(X)
    
    n = количество столбцов в X
    P = вектор из n нулей
    
    m = количество примеров
    
    // Градиентный спуск
    ДЛЯ i ОТ 1 ДО num_iterations:
        прогноз = X * P
        ошибки = прогноз - y
        градиент = (1 / m) * (X^T * ошибки)
        P = P - learning_rate * градиент
    КОНЕЦ ДЛЯ
    
    ВЕРНУТЬ P
КОНЕЦ ФУНКЦИИ
